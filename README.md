# Projeto CENSO

## instale o json-server e o serve pra executar o banco de dados e testar o build em produção

### npm install -g json-server -> Instalação global

### npm install --g serve -> instalação do serve global

## Inicie o server com o json-server, dentro de `src/api/` e execute *json-server --watch api.json*

## Baixe o projeto e inicie
```
npm install
```
